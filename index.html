MISUNDERSTOOD is a web app that takes a random New Yorker cartoon and captions it with a random line of poetry

The application accesses two different APIs, New Yorker's random cartoon and PoetryDB's poetry database.

The result is supposed to be a playful mixture of the absurd while often surprisingly relevant

MISUNDERSTOOD is built in HTML, CSS, and JS. The Axios library is used for remote API requests



Select RANDOM for a random combination

Select USER GENERATED to make your own caption and see how it may pertain to a random cartoon

Feel free to save your favorite combination locally and see it at FAVORITE
